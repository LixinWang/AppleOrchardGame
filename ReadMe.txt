CS 2261 HW03 GAME BY LIXIN WANG
APPLE ORCHARD

OBJECTIVE:
Catch as many good (red) apples as you can with the basket! Avoid the bad (black) apples or you will die and the game will be over!

Each red apple caught = +1 to score!

CONTROLS:
RIGHT ARROW KEY = moves the basket to the right
LEFT ARROW KEY = moves the basket to the left

Press START on the home screen to begin the game.
Press START on the ‘game over’ screen to return to the home screen.
